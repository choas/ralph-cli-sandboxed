# ralph-cli-sandboxed

A CLI tool for Ralph.

## Installation

```bash
npm install -g ralph-cli-sandboxed
```

## Usage

Run the CLI:

```bash
ralph
```

### Options

- `--version`, `-v`: Display the current version

```bash
ralph --version
```

## Development

### Prerequisites

- Node.js
- npm

### Setup

```bash
npm install
```

### Build

```bash
npm run build
```

### Type Check

```bash
npm run typecheck
```

### Run Tests

```bash
npm test
```
